# Fitbit Fitness Tracker Data Analysis

## Overview
A comprehensive analysis of Fitbit tracker data to uncover insights into user activity, sleep, and health metrics. Includes predictive modeling for calorie expenditure.

## Project Structure
- `src/`: Source code modules.
- `data/`: Dataset storage.
- `models/`: Trained ML models.
- `plots/`: Generated visualizations.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the pipeline:
   ```bash
   python main.py
   ```
